## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ yarn
```

## Migrating database

```bash
# Make migration
$ yarn migrate:make

# Run all migration
$ yarn migrate:latest

# Rollback all migration
$ yarn migrate:rollback
```

## Running the app

```bash
# development
$ yarn dev

# production mode
$ yarn start
```
